# Img2flag                                                         


# Install :

1. ```sudo apt install tesseract-ocr libtesseract-dev -y```

2. ```git clone https://github.com/cyberpoul/img2flag.git```

3. ```cd img2flag```

4. ```pip install -r requirements.txt --break-system-packages```


# Usage :
* ```python img2flag.py -f flag.png```
